# `agenthub ls`

Use this command when the user wants to browse the public Agent Hub roots or a path under them.

## Notes

- `agenthub ls` lists the public roots: `profile/`, `memory/`, `project/`, `skill/`, `secret/`, `platform/`.
- A leading `/` is optional.
- Prefer `agenthub ls` over old `agenthub list` or `agenthub files ls` phrasing.
