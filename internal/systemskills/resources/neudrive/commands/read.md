# `neudrive read`

Use this command when the user wants the contents or summary of one neuDrive path.

## Examples

- `neudrive read profile/preferences`
- `neudrive read project/demo`
- `neudrive read skill/writer/SKILL.md`
- `neudrive read secret/auth.github`

## Notes

- `project/<name>` is a summary view.
- A leading `/` is optional.
