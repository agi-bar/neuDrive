# `agenthub read`

Use this command when the user wants the contents or summary of one Agent Hub path.

## Examples

- `agenthub read profile/preferences`
- `agenthub read project/demo`
- `agenthub read skill/writer/SKILL.md`
- `agenthub read secret/auth.github`

## Notes

- `project/<name>` is a summary view.
- A leading `/` is optional.
