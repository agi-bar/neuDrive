# `neudrive ls`

Use this command when the user wants to browse the public neuDrive roots or a path under them.

## Notes

- `neudrive ls` lists the public roots: `profile/`, `memory/`, `project/`, `skill/`, `secret/`, `platform/`.
- A leading `/` is optional.
- Prefer `neudrive ls` over old `neudrive list` or `neudrive files ls` phrasing.
