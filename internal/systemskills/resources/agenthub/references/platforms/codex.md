# Codex Entry

- Local entrypoint: `$agenthub <subcommand>`
- Alternative discovery path: `/skills` and select `agenthub`
- Use `export` to capture Codex-visible data into Agent Hub.
- Use `import` to restore Agent Hub material back into Codex-compatible files and notes.
- Use `list` to inspect supported domains and discovered sources.
- Use `status` to verify MCP, skill install, and daemon readiness.
