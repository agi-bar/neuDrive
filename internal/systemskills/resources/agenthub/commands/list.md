# `agenthub list`

Use this command when the user wants to see what can be synchronized between the current platform and Agent Hub.

## Show

- Supported data domains
- Discovered local sources
- Current MCP connectivity
- Installed Agent Hub entrypoint status
- Any portability gaps or unsupported categories

## Notes

- Prefer concise summaries first.
- Expand to full file or directory listings only when the user asks for detail.
